Changelog

v0.3.1 (2022/07/22)
-------------------

* Added support for even earlier beta releases

v0.3.0 (2022/07/20)
-------------------

* Added support for (some) beta releases

  as a library in typical Spigot fashion

v0.1.0 (2022/07/16)

* Initial release.
